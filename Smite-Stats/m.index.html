Recent News

Patch 9.5 Balance Update [5/22/22] V2.1.2

Admittedly this update arrived later than I initially intended. The only things that were changed were the health numbers and normal changes included in V9.5 of SMITE. But I also wanted to make a point to mention that additions for the site will be halted for a brief time while I focus on revamping the UI. Currently, the site is functional on all relevant browsers, but with visual glitches that are likely to get worse in the future. To rememdy this, I'm hoping to switch over to React.js. Please be patient while these changes take place. This will NOT affect SMITE updates and I will continue to try to add every patch as soon as possible.

Quality of Life Update [4/30/22] V2.1

This update adds the following QOL features:

- Seperate 'Penetration' and 'Percent Penetration' into two different stat bars.
- Removed 'Armour Reduction' stat, still internally exists.
- Shortened links for link sharing.
- Added better search (mostly affects PC).
- Item info now persists to make checking an items information, again, easier.

As well as the following feature:

- Added 'Randomizer' button (PC exclusive).

I'm aware of the clear decline in quality on the mobile page of this website, a future update is planned to address that by being dedicated to revamping the mobile version.

Massive Feature Update [4/21/22] V.2

This major update adds and changes the following:

- Added visible stat cap gradient bars.
- Removed HP5 and MP5 consideration
- Added 'Console Window' which can be opened by using the ~ key.
- Fixed a bug with the delete item option not working as intended.
- Fixed issues with 'Glyph' items.
- Added all buffs, buffs use their Conquest statistics in calculation.
- Added build number toggle
- Added link sharing

Some things to note:

- Item passives and some god passives are still not here yet, so items like Transendence and Pendulum of Ages' mana conversion will have to wait until a future update. These passives will be toggle-able.
- Buff descriptions can be read by hovering over their icon on the God card.

Some Visual Changes [4/6/22] V1.2

This minor update changes the following:

- "Fire Giant Buffs" are now larger and thus more visible on the God card.
- God pantheons and roles now display with their respective icons.
- "News" tab replaced with "Info" tab, displays news and site information separately.
Major updates to come.

Website Released! [3/25/22] V1 - V1.0.1

SmiteStats is now available to the public! Not every update I wanted to release with is here, but they're coming. Including, but not limited to:

- Health Based Passives: Ability to Set Current Health & Mana
- Basic Attack Damage Comparisons

- Stacks
- More buffs!

Have suggestions? Email me at business@smitebuildmaker.com and I'll see what I can do!

About SmiteStats

What is SmiteStats?

SmiteStats is a personal fan-made project related to the video game SMITE that acts as a simulation of the in-game "god builder" with added features. It's design philosophy includes a single, downloadable webpage that's easy to access and cross-platform.

Want to support me? Donate!

Donations help me live and let me keep working on websites like this. If you wish to donate, you can do it here.

You can also follow me on Twitter @MichaelWarmbier

FAQ

Q: Is this website open source?
A: No. Suggestions are appreciated though, and I'm interested in working with contributers in the future.

Q: There's a bug with the website! How can I contact you?
A: You can contact me at business@smitebuildmaker.com. Keep in mind that you can also view the website console via inspect element at any time to see what's going on.

Q: The numbers are different than the in-game builder, what gives?
A: The in-game builder may round slightly different or not include certain passives (which will be clearly shown in a future update). Still, if you feel there is a significant issue, contact me and I'll get on it right away.

Q: Does this site work on both mobile and PC?
A: Yes!

Q: The website was updated without patch notes, why?
A: The format for the patch follows the following format: Major_Update.Minor_Update.Patch_Update. Patch updates and some bug fixes are not included in the patch notes.
